# Restricciones del proyecto

## Objetivo:
Desarrollar un chatbot RAG para madres gestantes que clasifique intención, detecte riesgo clínico y genere respuestas fundamentadas en los datasets del proyecto.

## Hardware disponible:

CPU AMD Ryzen 5
GPU NVIDIA RTX 2050
RAM: 16 GB RAM

## Tecnologías preferidas:

Python
FastAPI
LangChain
FAISS
STREAMLIT
API de Groq

# Restricciones:

No utilizar QLoRA en esta fase.
No realizar fine tuning.
Priorizar simplicidad y rapidez de implementación.
Mantener costos cercanos a cero.
Los datasets deben utilizarse completos según indicación del proyecto.

# Entregable esperado:

MVP funcional
Clasificación de intención
Clasificación de riesgo
Recuperación RAG
Interfaz conversacional
Evaluación básica
